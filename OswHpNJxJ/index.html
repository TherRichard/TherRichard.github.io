Go语言

2022-09-05 1 min read

一、先更新
apt-get update
apt-get -y upgrade ####这个应该不用，也能吧
二、下载Go源码编译
wget + Go语言官网Linux版本的地址
tar -xvf go1.9.3.linux-amd64.tar.gz
mv go /usr/local
三、设置Go系统环境变量
export GOROOT=/usr/local/go
export PATH=GOPATH/bin:GOPATH/bin:GOPATH/bin:GOROOT/bin:$PATH
四、此时可以查看是否安装成功
go version
